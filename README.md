# datatype range

| Type      | Typical Bit Width | Typical Range                   |
| :-------- | :-----------------| :-----------------------------: |
| char      | 1byte             | -128 to 127                     |
| short int | 2byte             | -32,768 to 32,767               |
| int       | 4byte             | -2,147,483,648 to 2,147,483,647 |
