# Abstract

convex hull algorithm들중 graham scan algorithm에 대해 정리한다.

# Idea

필드에 꽃혀 있는 무수히 많은 압정들을 생각해 보자. 고무줄을 하나
구해서 제일 아래쪽의 압정에 걸어서 모든 압정들을 둘러싸게 한바퀴 돌려보자.
고무줄은 모든 점들을 둘러싸게 되고 이것이 바로 convex hull이다.
